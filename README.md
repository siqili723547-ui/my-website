# 个人作品集网站

一个基于纯 HTML/CSS/JavaScript 的现代个人作品集网站，采用创意动效设计。

## 特性

- **创意动效**: 滚动触发动画、打字机效果、卡片悬停 3D 效果
- **响应式设计**: 完美适配桌面、平板、手机
- **零依赖**: 无需构建工具，直接在浏览器运行
- **易于部署**: 一键部署到 GitHub Pages

## 快速开始

### 本地预览

用浏览器直接打开 `src/index.html`

或使用 Python 启动本地服务器：

```bash
cd src
python -m http.server 8000
```

### 编辑内容

编辑 `content/` 目录下的 JSON 文件即可更新网站内容
